###2、CMake was unable to find a build program corresponding to "Ninja"

1、打开终端，然后输入

ls -an

查看".bash_profile"文件是否存在，如果存在就跳过第2步


2、创建环境变量配置文件

touch .bash_profile


3、打开并编辑配置文件

open .bash_profile

添加ninja的path路径：

~/Library/Android/sdk/cmake/3.10.2.4988404/bin

4、使配置生效
  
  source .bash_profile
  
  
5、查看环境变量
  
  echo $PATH
  
6、重启电脑
